# H-A-Q Marg – Women Legal Support Platform

## Project Overview

**H-A-Q Marg** is a web platform designed to help women understand their legal rights and access guidance related to family law issues.

The platform provides an **AI-assisted legal chatbot, legal case information, and petition guidance** so that women can quickly learn about their rights in situations like divorce, domestic violence, child custody, and financial maintenance.

The goal of the project is to **make legal awareness simple, accessible, and instant**, especially for users who may not have immediate access to legal professionals.

---

# Problem Statement

Many women are unaware of their **legal rights and available protections under Indian law**.

Legal information is often:

* Complex
* Hard to access
* Written in technical legal language

This project simplifies legal information and provides **instant digital assistance**.

---

# Solution

H-A-Q Marg provides a **simple web interface** where users can:

1. Chat with an **AI Legal Assistant**
2. Explore **important legal rights and laws**
3. Learn about **divorce, domestic violence, maintenance, custody**
4. Generate **basic legal petition formats**

---

# Key Features

### 1. AI Legal Assistant

Users can type their legal problem and receive guidance based on predefined legal knowledge.

Example queries:

* "I want divorce"
* "My husband is abusing me"
* "Who will get custody of my child?"
* "My husband is not giving me money"

---

### 2. Know Your Rights

Displays legal rights and protections related to women's safety and family law.

---

### 3. Petition Generator

Helps users understand **how legal petitions are structured** for filing cases.

---

### 4. Clean and Simple UI

The platform uses a **Royal Purple themed UI** to maintain a professional and accessible interface.

# Project Setup

## 1. Clone the Repository

```bash
git clone https://github.com/yourusername/H-A-QMarg.git
```

---

## 2. Navigate to Project Folder

```bash
cd H-A-QMarg
```

---

## 3. Install Backend Dependencies

```bash
npm install
```

---

## 4. Start Backend Server

```bash
node server.js
```

Server will run on:

```
http://localhost:3000
```

---

## 5. Run Frontend

Open:

```
Frontend/index.html
```

in your browser.
